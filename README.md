# Sector Breadth Divergence Dashboard

Live dashboard tracking S&P 500 sector breadth divergence across 7 moving average timeframes.

## Live Site
https://sofiatrade.github.io/breadth-signal

## What It Measures
Standard deviation of % stocks above MA across all 11 S&P sectors.
High σ = sectors diverging = hidden weakness.
Low σ = sectors moving together = healthy trend.

## Sectors Tracked
XLK, XLV, XLF, XLI, XLP, XLU, XLY, XLE, XLB, XLRE, XLC

## Timeframes
3D, 5D, 10D, 20D, 50D, 100D, 200D

## Signal Levels
- σ > 2.0 = DANGER
- σ > 1.5 = WARNING
- σ > 1.0 = CAUTION
- σ < 0.5 = CALM

## Data Source
MarketCharts.com — PCTASMA indicators per sector
